    Receita

    O projeto de receita foi o primeiro projeto que criamos, foi nele que começamos a entender o que é e para que serve o HTML. Ele é composto por uma série de elementos, que são tags ou marcações especiais inseridas no código para definir diferentes partes da página. Cada elemento possui uma função específica e contribui para a organização e formatação do conteúdo.

    Curriculo

    Após entender o que era o HTML começamos a aprender o CSS, que é uma linguagem utilizada para estilizar e formatar páginas da web escritas em HTML. Enquanto o HTML é responsável pela estrutura e conteúdo da página, o CSS controla a aparência visual, o layout e o design. Entendendo isso, desenvolvemos o projeto do Curriculo, que foi a implementação do HTML e CSS.
