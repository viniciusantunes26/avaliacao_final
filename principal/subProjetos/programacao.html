HTML, CSS, JavaScript

    HTML

    HTML é uma linguagem de marcação usada para criar a estrutura e o conteúdo de uma página da web. Ele fornece os elementos básicos necessários para definir a estrutura de uma página, como cabeçalhos, parágrafos, imagens, links e listas. O HTML usa tags para marcar e descrever cada elemento na página, permitindo que os navegadores da web interpretem e exibam corretamente o conteúdo.


    CSS

    CSS é uma linguagem de estilo usada para controlar a aparência e o layout de uma página da web. Ele trabalha em conjunto com o HTML, fornecendo um conjunto de regras que determinam como os elementos HTML devem ser exibidos. Com o CSS, você pode definir propriedades como cor, fonte, tamanho, margens, posicionamento e muito mais. O CSS permite separar o conteúdo (HTML) da apresentação (estilos), o que facilita a manutenção e a atualização do design de um site.


    JavaScript

    JavaScript é uma linguagem de programação de alto nível que é amplamente utilizada para adicionar interatividade e funcionalidade a páginas da web. Ao contrário do HTML e do CSS, que são usados para estrutura e estilo, respectivamente, o JavaScript permite que você crie comportamentos dinâmicos em uma página.

    Com o JavaScript, você pode manipular elementos HTML, responder a eventos do usuário (como cliques e rolagem), validar formulários, criar animações, interagir com APIs externas e muito mais. Ele é executado diretamente no navegador da web do usuário, permitindo que você crie experiências interativas e responsivas.

    O JavaScript é uma linguagem versátil e possui uma ampla gama de recursos, como suporte a orientação a objetos, manipulação de strings e arrays, expressões regulares, funções assíncronas e muito mais. Ele também é utilizado além do desenvolvimento web, sendo adotado em aplicações de backend, criação de jogos e desenvolvimento de aplicativos móveis.

    Receita

    Esse foi o primeiro projeto criado em sala de aula, nele começamos a entender o que é HTML e como ele funciona.
    Ele foi criado somente no HTML sem a utilização de CSS para estilização. Assim aprendemos que existem varias estruturas fundamentais chamadas de Tags para a criação de uma página web como:
    • <html> : É o elemento raiz que envolve todo o conteúdo HTML de uma página.
      <head> : Contém informações sobre a página, como título, metadados, links para estilos CSS e scripts JavaScript.
      <body> : Contém o conteúdo visível da página, como texto, imagens, links, elementos interativos, etc. É dentro do que a maior parte do conteúdo é colocada.
      <header> : Representa o cabeçalho da página ou de uma seção. Geralmente contém elementos como logotipos, títulos e navegação.
      <main> : Representa o conteúdo principal da página. Deve ser único em uma página e normalmente contém o conteúdo central.
      <footer> : Representa o rodapé da página ou de uma seção. Pode conter informações de contato, direitos autorais e links adicionais.

    Manipulando DOM

    Ainda com o foco em JavaScript, o DOM cria uma árvore de objetos que representa a estrutura hierárquica dos elementos HTML de uma página da web. Cada elemento HTML é representado por um objeto no DOM, e esses objetos podem ser manipulados por meio do JavaScript para realizar tarefas como adicionar, modificar ou excluir elementos e seus atributos.
    O DOM em JavaScript é uma interface padronizada e independente de plataforma para manipular documentos HTML. Ele nos permite acessar e manipular dinamicamente a estrutura e o conteúdo de uma página da web. Por exemplo, você pode usar o DOM para adicionar um novo elemento HTML à página, alterar o texto de um parágrafo existente, alterar o estilo de um elemento, criar eventos interativos, entre muitas outras possibilidades.
