Banco de Dados

    Banco de Dados

    Um banco de dados é uma coleção organizada de dados que são armazenados eletronicamente em um sistema de computador. Ele é projetado para armazenar, gerenciar e recuperar informações de forma eficiente e confiável. Os bancos de dados são amplamente utilizados em diversos domínios, como empresas, instituições governamentais, pesquisa científica, comércio eletrônico e muitos outros. Eles são essenciais para o armazenamento e acesso rápido a grandes volumes de informações estruturadas.


    Exercício 1

    Existem diferentes tipos de diagramas que podem ser usados para representar a estrutura e as relações em um banco de dados, como por exemplo o Diagrama Entidade-Relacionamento (DER): O diagrama Entidade-Relacionamento é amplamente utilizado para modelar a estrutura lógica de um banco de dados. Ele representa as entidades (tabelas), os relacionamentos entre as entidades e os atributos de cada entidade. Nesse diagrama, as entidades são representadas por retângulos, os relacionamentos por linhas e os atributos por elipses.

    Exercício 2

    No MER, as entidades são objetos do mundo real ou conceitual que possuem características distintas e relacionamentos com outras entidades. As entidades são representadas por retângulos no diagrama do Modelo Entidade-Relacionamento. Cada entidade tem atributos, que são as propriedades ou características que descrevem a entidade. Os atributos são representados dentro do retângulo da entidade.

    Atomicidade



    Seria a garantia de que a transação será feita totalmente ou não será feita. Nesse caso, a transação não é feita “pela metade”. Se por ventura uma operação da transação falhar, consequentemente, toda a transação falhará.

    Consistência

    Seria a proteção da integridade dos dados. Ou seja, se um banco de dados fizer uma operação que não seja válida, o processo será impedido e retornará para o estado inicial do processo.

    Isolamento

    A capacidade de isolamento seria o fato de uma transação não “atrapalhar” a outra e ocorrer de forma isolada, garantindo que sejam feitas de forma individual.

    Durabilidade

    Seria a preservação dos dados após as operações terem sido realizadas. Ou seja, uma vez que uma transação for efetuada, ela permanecerá dessa forma, mesmo que ocorram problemas graves no sistema, sem precisar de retrabalho.
