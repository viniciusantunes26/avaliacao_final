    Automação Industrial

    A automação industrial é um campo de estudo e aplicação que envolve o uso de tecnologia para controlar e monitorar processos industriais de forma automatizada. Essas técnologias estão cada vez mais ligadas com a área de T.I. trazendo ainda mais a otmização da produção industrial, aumento da eficiência, redução custos e melhoraria da qualidade dos produtos.

    A automação industrial abrange diversos conceitos e princípios fundamentais. Um dos elementos centrais é o sistema de controle, que consiste em um conjunto de dispositivos, sensores e atuadores que nescessitam de uma comunicação e de um armazenamento de dados em uma central, assim permitem a supervisão e regulação de processos industriais.


    Sensor Ultrassônico

    Durante as aulas de automação criamos alguns projetos que demonstram a interação da área de técnologia da informação com a área industrial. Um desses projetos foi programar um sensor Ultrassônico, que usa a linguagem C como uma maneira de "dizer" ao sensor como ele deve funcionar.
    Nesse projeto foi utilizado lógicas de programação para definir uma distância dentro do limite de alcance do sensor e para identificar um objeto que esteja entre o sensor e o limite, assim podendo comunicar a uma outra parte da produção, para que seja realizado uma outra ação.

    Sequência de Led's

    O projeto de Led's foi criado para que entendessemos na pratica como funciona a estrutura de controle "for", mostrando a inicialização de uma variável, a condição que é a expressão lógica que determina se o loop deve continuar sendo executado ou não, e o incremento, que é a parte em que se especifica como a variável de controle será modificada a cada iteração do loop.
    A plataforma Tinkercad foi utilizada para a criação desses circuitos e programas, nela montamos os circuitos componentes por componentes, e digitamos todo o codigo que será executado dentro do microcontrolador, ao finalizar a própria plataforma cria um circuito eletrônico do que foi criado.

    Circuito do Farol

    O circuito do farol foi desáfio passado, para que nós criassemos três faróis sendo eles dois faróis para carros e um para pedestres. A intenção desse desáfio foi desenvolver lógicas de progrmação usando estruturas de controles e funções.
