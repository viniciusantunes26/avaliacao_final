    Receita

    Esse foi o primeiro projeto criado em sala de aula, nele começamos a entender o que é HTML e como ele funciona.
    Ele foi criado somente no HTML sem a utilização de CSS para estilização. Assim aprendemos que existem varias estruturas fundamentais chamadas de Tags para a criação de uma página web como:
    • <html> : É o elemento raiz que envolve todo o conteúdo HTML de uma página.
      <head> : Contém informações sobre a página, como título, metadados, links para estilos CSS e scripts JavaScript.
      <body> : Contém o conteúdo visível da página, como texto, imagens, links, elementos interativos, etc. É dentro do que a maior parte do conteúdo é colocada.
      <header> : Representa o cabeçalho da página ou de uma seção. Geralmente contém elementos como logotipos, títulos e navegação.
      <main> : Representa o conteúdo principal da página. Deve ser único em uma página e normalmente contém o conteúdo central.
      <footer> : Representa o rodapé da página ou de uma seção. Pode conter informações de contato, direitos autorais e links adicionais.

    Curriculo

    Após entender o que era o HTML começamos a aprender o CSS, que é uma linguagem utilizada para estilizar e formatar páginas da web escritas em HTML. O CSS controla a aparência visual, o layout e o design. Entendendo isso, desenvolvemos o projeto do Curriculo, que foi a implementação do HTML com o CSS.
